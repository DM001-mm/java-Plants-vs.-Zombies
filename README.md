# Plants vs Zombies - Java

## 项目结构

```
com.xhl.pvz
├── Main.java
├── app/         # 应用层 (窗口、面板)
├── core/        # 核心 (游戏循环、状态、配置)
├── scene/       # 场景 (主菜单、关卡、暂停、结束)
├── entity/      # 实体 (植物、僵尸、子弹、物品)
├── lawn/        # 草坪 (网格、单元格、行)
├── ui/          # UI (按钮、卡片、阳光)
├── input/       # 输入 (鼠标、键盘)
├── event/       # 事件 (事件总线)
├── command/     # 命令 (撤销/重做)
├── manager/     # 管理器 (资源、碰撞、关卡)
├── animation/   # 动画
├── save/        # 存档
└── util/        # 工具类
```
